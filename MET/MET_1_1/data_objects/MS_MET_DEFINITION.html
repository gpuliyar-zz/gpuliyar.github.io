Rising/Dropping Indicator?
Rising_Dropping Indicator

MS_MET_Common_Lov_Infolet
(KI Rise Drop Indicator) 		Use this field to indicate the trending patterns of KPI/KRI. Based on the selection, appropriate fields will be visible at the Overall Thresholds section, allowing the user to provide numeric values to the corresponding fields. This can be further used for reporting purposes. For the Rising Indicator, if the actual value entered in the KPIKRI Data Entry form exceeds the critical or non-critical thresholds, then the KPIKRI has breached the threshold and email notification will go out. For the Dropping Indicator, if the actual value entered in the KPIKRI Data Entry form goes below the critical or non-critical thresholds, then the KPIKRI has breached the threshold and email notification will go out. If the Indicator is set to Both, then if the actual value entered in the KPIKRI Data Entry form goes below the critical or non-critical thresholds, then the KPIKRI has breached the threshold and email notification will go out.

EDITOR_OWNERSHIP_CHECK

public VARCHAR2 EDITOR_OWNERSHIP_CHECK(PC_FLOW_CODE IN VARCHAR2, 
                                       PC_STAGE_CODE IN VARCHAR2, 
                                       PN_PID IN NUMBER, 
                                       PN_ID IN NUMBER)
Returns one of the below.
Parameters:
PC_FLOW_CODE - Flow Code from the DD Process code in the Form
PC_STAGE_CODE - Stage code from the Current Stage.
PN_PID - process instance id of the form
PN_ID - instance id of the form
Returns:
Returns one of the below:
OWNER : If the user making the edit is the owner of the object
NOT_OWNER : If user is not a owner, and there are other owners
NO_USERS : There are no owners for this object

COPY_AUDIT_PROC

public  COPY_AUDIT_PROC(LT_IN_DATA_ORIG IN CLOB, 
                        LT_IN_DATA IN CLOB, 
                        PC_PROCESS_CODE IN VARCHAR2, 
                        LN_ORIG_PROCESS_INSTANCE_ID IN NUMBER, 
                        LN_PROCESS_INSTANCE_ID IN OUT NUMBER, 
                        XN_ERROR_HANDLE_ID IN OUT NUMBER, 
                        XN_ERROR_SEQ IN OUT NUMBER, 
                        XN_ERROR_CODE OUT VARCHAR2)
This hook procedure is invoked on submitting the Copy Audit form. This is invoked in T001 self-transition from S001 to S001 stage. It picks up the details of new audit from the copy audit xml, generates an xml for the old audit whose information is to be copied to form new audit and updates the xml for old audit with the new data from copy audit xml. After the enrichment of xml it triggers the F003 flow which eventually creates the new audit
Parameters:
LT_IN_DATA_ORIG - holds the input-data passed by the calling environment
LT_IN_DATA - holds the manipulated input-data passed by the calling environment
PC_PROCESS_CODE - Workflow code
LN_ORIG_PROCESS_INSTANCE_ID - Process Instance ID to be updated with
LN_PROCESS_INSTANCE_ID - Process Instance ID to be updated with
XN_ERROR_HANDLE_ID - XML error handle id
XN_ERROR_SEQ - XML error sequence
XN_ERROR_CODE - XML error code

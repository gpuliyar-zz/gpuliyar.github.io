Form Name PLAN_SCOPE
Table Name MS_AUD_PLAN_SCOPE
Secondary Title Type Field
Secondary Title Source AUDIT_TITLE
Form Actions Field
Status Field
Action Field ACTIONS
Security Access Infolet MS AUD Access Control Infolet
Description This form is used to update the scope of the Audit i.e., to perform detailed scoping of the audit. This form can be accessed from the My Audits infoport report. By default this form can be edited by the Lead Auditor, Audit Manager and auditors of the Audit.

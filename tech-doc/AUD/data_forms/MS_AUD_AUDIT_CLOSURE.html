Description The Lead Auditor or Audit Manager can close the Audit by initiating the Audit Closure process. This form can be accessed through Actions gear icon in My Audits infoport output report. The Lead Auditor and Audit Manager need the AUD - Close Audit activity to perform audit closure process.

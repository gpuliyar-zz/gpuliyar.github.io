CHECK_FORM_ACCESS

public NUMBER CHECK_FORM_ACCESS(IP_PUSHFORM_ID IN NUMBER, 
                                IP_PROCESS_INSTANCE_ID IN NUMBER, 
                                IP_USER_NAME IN VARCHAR2)
This function will return 0 if user [who tries to access any data form] does not have proper access/privilege to work on or view details, returns 1 if user has write access else return 9 if user have read access.
Parameters:
IP_PUSHFORM_ID - take input as pushform metric id
IP_PROCESS_INSTANCE_ID - take input as current process instance id
IP_USER_NAME - take input of user name who try to access the form

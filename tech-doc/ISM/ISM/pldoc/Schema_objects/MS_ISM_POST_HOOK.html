POST_HOOK

public  POST_HOOK(LT_IN_DATA_ORIG IN CLOB, 
                  LT_IN_DATA IN CLOB, 
                  PC_PROCESS_CODE IN VARCHAR2, 
                  LN_ORIG_PROCESS_INSTANCE_ID IN NUMBER, 
                  LN_PROCESS_INSTANCE_ID IN OUT NUMBER, 
                  XN_ERROR_HANDLE_ID IN OUT NUMBER, 
                  XN_ERROR_SEQ IN OUT NUMBER, 
                  XN_ERROR_CODE OUT VARCHAR2)
This procedure is mainly used to update latest PID and Instance ID into both Issue and Action MS tables. Also used to sending notification email to action owener when open action are uploaded at monitor stage and send auto delegation email.
Parameters:
LT_IN_DATA_ORIG - contain input xml data in clob format
LT_IN_DATA - return updated xml data in clob format
PC_PROCESS_CODE - contain workflow process code
LN_ORIG_PROCESS_INSTANCE_ID - contain old process instance id
LN_PROCESS_INSTANCE_ID - contain updated process instance id
XN_ERROR_HANDLE_ID - return error number
XN_ERROR_SEQ - return error sequence number
XN_ERROR_CODE - return error code

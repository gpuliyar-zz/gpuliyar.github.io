#4: Page Click

Page Click

Tree directory enables to traverse you to the desired page. The clicked page displays in the right content frame.

#5: Search Menu Engine

Search Item menu

It searches the keyword in the tree directory and expands till the keyword is found with a highlighted brown colour font.

#6 : Data Table Information

Data Table

1. Displays the object name 2. Enables to set number of row to be displayed 3. Searches the data within the table
4. Sorts the column(ascending/descending) 5. Displays which set of page is viewed 6. Page navigation

#7 : Navigation Link

Href Link

When href link is clicked, it navigates the page to that particular object.

#8 : Collapse Feature

Collapse tab

When Collapse is clicked, it opens the click tab and collapses other tab that are opened.

#9 : Code viewer

Code Viewer

When Code related data button is clicked, it opens in a formatted text inside modal window. Select All button is to copy to clipboard.

MS GRC REPORT PERIOD TYPE

Table Name MS_GRC_REPORT_PERIOD_TYPE
Description To data table is used to determine the logic for reporting period

  • Columns
  • Data
# Title
Name		 Type Description
1 Period Type ID
PERIOD_TYPE_ID 		Number(10) identifier for a reporting period typeb
2 Period Type
PERIOD_TYPE 		Varchar2(255 char) this can be defined by the customer based on all the different Period Categories they would require to configure in the system.
3 Applies To Organizations
APPLIES_TO_ORGANIZATIONS 		Varchar2(4000 char) this column would ensure that only the users belonging to the organizations listed here would be able to view the Reporting Period Type that it is linked with. This field logic would not be a part of the first release for Surveys.
4 Valid From
VALID_FROM 		Date data columns would be used to define the validity of a reporting period. A reporting period type wouldn’t show in the form if the current date is before the Valid from date.
5 Valid Until
VALID_UNTIL 		Date data columns would be used to define the validity of a reporting period. A reporting period type wouldn’t show in the form if the current date has exceeded the Valid Until date.
6 Locale_ID
Locale_ID 		Number(10)

GET_CACHED_CHILD_ORGS

public MS_GRC_VARCHAR2_ARRAY GET_CACHED_CHILD_ORGS(ORG_ID IN MS_GRC_VARCHAR2_ARRAY, 
                                                   UNAME IN VARCHAR2)
This function is used to check if the cache exist for username and Org IDs or not for view activity. If yes then it will directly fetch the values from cache otherwise it will call the procedure the store the value

UNAME - -- User name to be passed like 'Bella.Bader'

GET_ORGID_BY_USER_ACT_VIEW

public MS_GRC_VARCHAR2_ARRAY GET_ORGID_BY_USER_ACT_VIEW(PI_USER IN VARCHAR2, 
                                                        OBJ_TYPE IN VARCHAR2, 
                                                        OBJECTIVE_TYP IN VARCHAR2)
This function is used to check if the cache exist for username and object type or not for viewall activity. If yes, then it will not call procedure cache_user_act_org_viewall as it will directly fetch the value from cache else it will call the above procedure

OBJECTIVE_TYP - -- Objective type used in OBJECTIVE obejct only

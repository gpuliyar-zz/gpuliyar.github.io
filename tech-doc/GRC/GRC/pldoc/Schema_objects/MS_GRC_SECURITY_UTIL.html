LVC_VIEW_ACTIVITY - -- Calculate the view activity and pass
LVA_ORG_ID_ARR - -- To get all the Owner Orgs + Its parent Organizations IDs (Own Org id will be passed from external)
APP_TO - -- Calculate the APPLICABLE_TO either it should be null,1 or 2. If it is null treating it as 1

public VARCHAR2 GET_OBJECT_ACCESS_FOR_USER(OBJ_TYPE IN VARCHAR2, 
                                           OBJ_ID IN VARCHAR2, 
                                           OWNER_ORGS IN VARCHAR2, 
                                           REST_ACCESS_TO IN VARCHAR2, 
                                           PROFILE_USER_NAME IN VARCHAR2, 
                                           OBJECTIVE_TYP IN VARCHAR2 DEFAULT NULL)
This function is used to check if the Logged in user has access or not for the given Object ID and other parameters based on view activity.
Parameters:
OBJ_TYPE - -- Passed the Object Type
OBJ_ID - -- Passed the Object IDs
OWNER_ORGS - -- Passed the Owner Organization IDs
REST_ACCESS_TO - -- Pass its value (N,O,A)
PROFILE_USER_NAME - -- User name in varchar
OBJECTIVE_TYP - -- Pass the objective type. It is not a compulsary field

GET_OBJECT_LIST_BY_USER

public MS_VARCHAR2_ARRAY GET_OBJECT_LIST_BY_USER(USERNAME IN VARCHAR2, 
                                                 OBJTYP IN VARCHAR2)
This function is used to get all the OBJECT IDs for a given username and an object type. This will return the Object IDs based on the VIEW or VIEW ALL activities assigned to the user. This will be used for all the GRCF objects except EXCEPTION
Parameters:
USERNAME - -- User Name to be passed e.g. Lily.Lake
OBJTYP - -- Object type to be passed e.g. MS_GRC_PROCESS

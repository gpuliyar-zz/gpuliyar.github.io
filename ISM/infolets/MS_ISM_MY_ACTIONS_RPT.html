MS_ISM_MY_ACTIONS_RPT

Description This infolet provides the details of all the actions raised in the system where the logged in user is same as the action owner identified for those particular actions. All the actions that are assigned to the logged in user are to be pulled up in this report (including the actions that are in pending approval and pending review and closed stage).

MS_ISM_MY_ACTIONS_RPT Report SYSTEMI 
                                                        SELECT ms_ism_helper.
               get_link_url_for_my_actions ('ACTION',
                                            A.DD_PROCESS_INSTANCE_ID,
                                            A.ACTION_STATUS,
                                            A.ACTION_TITLE)  AS ACTION_TITLE,
       ms_ism_helper.get_status_display_name(A.ACTION_STATUS) AS ACTION_STATUS,
       MS_ISM_UTILITIES.GET_USER_FULL_NAME (A.ACTION_OWNER, '') AS ACTION_OWNER,
       B.ISSUE_TITLE AS ISSUE_TITLE,
       A.ACTION_DUE_DATE,
       B.ISSUE_DUE_DATE,
       (TO_DATE (SYSDATE, 'DD/MM/RRRR')- TO_DATE (A.ACTION_DUE_DATE, 'DD/MM/RRRR'))
          AS Days_past_due,
       A.NO_OF_EXTENSIONS AS NO_OF_EXTENSIONS
FROM MS_ISM_ACTION A, MS_ISM_ISSUE B
 WHERE A.ISSUE_ID = B.ISSUE_ID AND (A.ACTION_OWNER=:1 OR A.ACTION_DELEGATEE=:1)
AND A.ACTION_STATUS IN ('2','5','6','16','21')
 ORDER BY A.DD_CREATED_ON desc

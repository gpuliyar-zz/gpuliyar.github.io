MS RSM Get Monthwise Vacation String

Description This is attached to multiple fields (Freebusy Month2, freebusy Month3, Freebusy Month1) in RSM 6.0 Request form (MS_AUD_RSM_MANAGE_TEAM_RESOURCE) to get the comma separated hours for a particular time period related to absence and vacation from time sheet. How ever this form is deprecated in RSM 6.1 and new form has been introduced - MS_RSM_RES_REQUEST

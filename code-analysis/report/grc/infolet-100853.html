Infolet Details

Name MS_GRC_GET_ACTIONS (100853)
SQL 
SELECT distinct w.action_code as action_code,
  w.action_name as action_name,
  v.status_code as status_code
FROM ms_grc_status v,
  ms_grc_actions w,
  ms_grc_status_action_mapping x,
  si_users_t y,
  si_locales z
where y.user_name = :1
and w.locale_id   =
  (SELECT locale_id FROM si_locales WHERE locale_string=y.locale
  )
and w.action_code =x.actions
and V.STATUS_CODE =X.STATUS
and V.STATUS_CODE =:2
and nvl(current_Stage,:3) = :3
Description This infolet extracts data from multiple tables such as ms_grc_status and ms_grc_actions and is thereby used in all GRC library forms including Area of Compliance, Asset, Asset Class, Control, Evidence, Exception, Financial Account, Function, Framework Reference, Objective, Organizational Profile, Process, Product, Question/Procedure, Reference, Regulatory Body, Requirement, Risk and Standard

Name MS_GRC_GET_DATA_ASSERTIONS (100864)
SQL 
SELECT Displayed_Value, stored_value
FROM Ms_Qs_List_Of_Values Lovvalue,
  Ms_Qs_Lov_Names Lovname,
  Si_Ent Ent,
  si_users_t usr,
  si_locales loc
WHERE Lovname.Lov_Name     = 'MS_GRC_Assertions_Lov'
AND Lovvalue.Lov_Id        = Lovname.Lov_Id
AND Lovvalue.Org_Entity_Id = 1
AND Lovvalue.Enterprise_Id = Ent.Enterprise_Id
AND Lovname.Enterprise_Id  = Ent.Enterprise_Id
AND loc.Locale_string      = usr.LOCALE
AND loc.LOCALE_ID          = Lovvalue.locale_id
AND usr.USER_NAME          = :1 order by Displayed_Value
Description This infolet extracts data from Ms_Qs_List_Of_Values, Ms_Qs_Lov_Names, Si_Ent Ent, si_users_t and si_locales tables and is thereby used in MS GRC Control Infolet and MS_GRC_61_SP2_FIN_ACC_LIST_INF infolets

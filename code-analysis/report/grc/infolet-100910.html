Infolet Details

Name MS_GRC_GET_STATUS (100910)
SQL 
SELECT x.STATUS_NAME as STATUS_NAME,x.status_code as status_code
FROM Ms_GRC_Status x,
  Si_Users_t y
WHERE y.user_name    = :1
AND x.locale_id    =
  (select locale_id from si_locales where locale_string=y.locale
  )
And x.status_code=NVL(:2,x.status_code) and form_name is null
Description This infolet extracts data from Ms_GRC_Status table and is thereby used in all GRC library forms including Area of Compliance, Asset, Asset Class, Control, Evidence, Exception, Financial Account, Function, Framework Reference, Objective, Organizational Profile, Process, Product, Question/Procedure, Reference, Regulatory Body, Requirement, Risk and Standard

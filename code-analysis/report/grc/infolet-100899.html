Name MS_GRC_GET_ORGANIZATIONS (100899)
SQL 
SELECT org.org_entity_id   AS org_entity_id
, org.org_entity_name          AS org_entity_name
FROM si_org_entities org
, si_org_locations loc
, si_org_hierarchy_levels ohl
, si_org_entities porg
WHERE org.location_id = loc.location_id (+)
AND SYSDATE BETWEEN org.start_date AND Nvl(org.end_date, SYSDATE)
AND org.org_hierarchy_id = ohl.org_hierarchy_id
AND org.parent_org_entity_id = porg.org_entity_id (+)
Order By org.org_entity_name
Description This infolet extracts data from si_org_entities, si_org_locations, si_org_hierarchy_levels tables and is thereby used in some of the list report infolets

Name MS_GRC_TYPE_OF_EXCEPTION (101095)
SQL 
Select c.Exception_for_id,
       c.Exception_for_value
       From   ms_grc_exception_config c,si_locales a,si_users_t b
       where c.locale_id= a.locale_id and
       a.locale_string = b.locale 
       and b.user_name = :1
       order by c.Exception_for_value
Description This infolet extracts data from ms_grc_exception_config table and is thereby used in Exception form

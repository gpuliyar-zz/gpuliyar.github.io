Infolet Details

Name MS_GRC_Get_RQMT_Flt_Inf (100929)
SQL 
SELECT object_id AS Requirement_ID,
  object_name    AS Requirement_Name
FROM
  (SELECT OBJECT_ID ,
    OBJECT_NAME,
    DD_OBJECT_TYPE,
    (SELECT ms_concat (d.OWNER_ORGANIZATIONS)
    FROM ms_grc_requirement_org d
    WHERE a.object_id = d.object_id
    ) AS OWNER_ORGANIZATIONS,
    restrict_access_to,
    valid_from,
    valid_until
  FROM ms_grc_requirement a
  ) t1
WHERE dd_object_type                                                                                                      ='MS_GRC_REQUIREMENT'
AND ms_grc_utilities.ms_grc_SECURITY(t1.dd_object_type, t1.restrict_access_to, t1.OWNER_ORGANIZATIONS, :1, 'VIEW') = 'ACCESS'
AND TRUNC(sysdate) BETWEEN NVL(TRUNC(t1.valid_from),TRUNC(sysdate)) AND NVL(TRUNC(t1.valid_until),TRUNC(sysdate))
ORDER BY OBJECT_NAME ASC
Description This infolet extracts data from ms_grc_requirement_org and ms_grc_requirement tables and is thereby used in MS_GRC_61_SP2_AOC_LIST_INF, MS_GRC_61_SP2_ASSET_CLASS_LIST_INF, MS_GRC_61_SP2_ASSET_LIST_INF, MS_GRC_61_SP2_FUNCTION_LIST_INF, MS GRC Control Infolet, MS GRC Risk Infolet, MS_GRC_61_SP2_QUE_PROC_LIST_INF, MS GRC Process Rep Infolet, MS_GRC_61_SP2_EVI_LIST_INF, MS_GRC_61_SP2_STANDARD_LIST_INF, MS_GRC_61_SP2_REG_LIST_INF, MS_GRC_61_SP2_OBJ_REP_INF and MS_GRC_Exception_Report_Infolet

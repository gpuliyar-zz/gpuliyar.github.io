Infolet Details

Name MS_GRC_Get_AOC_Flt_Inf (100922)
SQL 
SELECT object_id AS Area_of_compliance_id,
  object_name    AS Area_of_compliance_Name
FROM
  (SELECT OBJECT_ID,
    OBJECT_NAME,
    dd_object_type,
    (SELECT ms_concat (d.OWNER_ORGANIZATIONS)
    FROM ms_grc_area_of_complia_ORG d
    WHERE a.object_id = d.object_id
    ) AS OWNER_ORGANIZATIONS,
    restrict_access_to,
    valid_from,
    valid_until
  FROM MS_GRC_AREA_OF_COMPLIA a
  ) t1
WHERE DD_OBJECT_TYPE                                                                                                      ='MS_GRC_AREA_OF_COMPLIANCE'
AND ms_grc_utilities.ms_grc_SECURITY(t1.dd_object_type, t1.restrict_access_to, t1.OWNER_ORGANIZATIONS, :1, 'VIEW') = 'ACCESS'
AND TRUNC(sysdate) BETWEEN NVL(TRUNC(t1.valid_from),TRUNC(sysdate)) AND NVL(TRUNC(t1.valid_until),TRUNC(sysdate))
ORDER BY OBJECT_NAME ASC
Description This infolet extracts data from MS_GRC_AREA_OF_COMPLIA_ORG and MS_GRC_AREA_OF_COMPLIA tables and is thereby used in MS_GRC_61_SP2_ASSET_CLASS_LIST_INF, MS GRC Control Infolet, MS GRC Risk Infolet, MS_GRC_61_SP2_QUE_PROC_LIST_INF, MS_GRC_Product_Rpt_Infolet, MS GRC Process Rep Infolet, MS GRC Requirement Infolet, MS_GRC_61_SP2_STANDARD_LIST_INF, MS_GRC_61_SP2_REG_LIST_INF and MS_GRC_61_SP2_OBJ_REP_INF infolets

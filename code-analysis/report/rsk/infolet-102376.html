Name MS_RSK_LND_NR_GET_RISKS (102376)
SQL 
SELECT DISTINCT b.RISK_ID,
  MS_RSK_UTILS.Fn_Get_risk_name(b.RISK_ID) AS RISK_NAME
FROM MS_RSK_RISK_ASSESSMENT A,
  MS_RSK_RISK_ASSESSMENT_RSK B
WHERE A.OBJECT_ID   = B.OBJECT_ID
AND RISK_ID   IS NOT NULL
AND MS_RSK_UTILS.Fn_Get_risk_name(b.RISK_ID) IS NOT NULL
Description The infolet is used to populate the Risks filter in the report Organizations at Risk (by individual Assessment)

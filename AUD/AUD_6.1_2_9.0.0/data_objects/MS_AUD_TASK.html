Final Approver
TS_FINAL_APPROVER

If Approvals Required is equal to By Lead Auditor and Audit Manager this field is visible and disabled. The Audit Managers name is populated in read-only mode. When Audit Classification is equal to Cross Functional the value is Any Approver this field is visible and editable. The task creator can select any approver who has the activity AUD - Approve tasks

Task Approver
TS_TASK_APPROVER

If Approvals Required = By Lead Auditor, this field is displayed in read-only mode with the Lead Auditors name. When the task owner sends this form for approval, the form sent to the Lead Auditor of the audit for approval If Approvals Required = By Lead Auditor and Audit Manager, then this field is displayed in read-only mode and the lead auditors name is displayed and the final approver field is visible in read-only mode If Approvals Required = By Any Approver the task creator can select any approver who has the activity AUD - Approve tasks who will approve the task. When Audit Classification = Cross Functional the value is Any Approver there will be 2 levels of approval where the task creator can select any approver who has the activity AUD - Approve tasks the Task Approver being mandatory the Final Approver being optional.

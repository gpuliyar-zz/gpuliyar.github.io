# Title
Name		 Type Description
1 Period ID
PERIOD_ID 		Number(10) identifier for a reporting period
2 Period Type
PERIOD_TYPE 		Number(10) this column should match all the values that have already been defined in the Reporting Period Type data table Period Type column.
3 Period Name
PERIOD_NAME 		Varchar2(255 char) this column would have period names corresponding to the Reporting Period Types defined in the previous table
4 Period Start Date
PERIOD_START_DATE 		Date these date columns would be used to define the start for a defined period. These dates would not be used for the current release of Surveys.
5 Period End Date
PERIOD_END_DATE 		Date these date columns would be used to define the end date for a defined period. These dates would not be used for the current release of Surveys.
6 Assignment Start Date
ASSIGNMENT_START_DATE 		Date these dates would determine which Period Name options would display for the user’s selection while setting up the survey
7 Assignment End Date
ASSIGNMENT_END_DATE 		Date these dates would determine which Period Name options would display for the user’s selection while setting up the survey.

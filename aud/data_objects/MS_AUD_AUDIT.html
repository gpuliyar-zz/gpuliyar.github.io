AUDIT_TYPE

MS_AUD_AUDIT_Dependent_List_details This field displays the value of the possible audit types. This field depends on the selected Audit Classification. If the Audit Classification is Compliance then the default values are Financial, Operations, Regulatory, Environmental, Health and Safety and SOX. If the Audit Classification is Cross Functional, then the default values here are Financial, Operations, Regulatory, Environmental, Health and Safety and SOX. If the Audit Classification is Quality , then the default values here are Process, Product and System. If the Audit Classification is Internal, then the default values here are Compliance, Financial, Investigation, IT/IS, Management, Operational, Performance and Tax. If the Audit Classification is Operational, then the default values here are Third Party, EHS, Labor, Product, Quality and Sustainability. If the Audit Classification is Integrated, then the default values here are Environmental, Financial, Health and Safety, Operations, Regulatory and SOX.

Scope
AE_SCOPE

The AEs that have come from the Plan form or Audit form (for adhoc audits) cannot be deleted and can only be set as Out of Scope, the system needs to validate if there are tasks created using this scope and if there are the system will not allow the user to set it as Out of Scope. In-scope will be the default value.

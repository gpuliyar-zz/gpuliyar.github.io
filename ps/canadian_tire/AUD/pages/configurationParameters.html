#1 : MS AUD Integrated Modules

Scope Enterprise
Description Define Parameter for integration with other Modules.

Parameter Name Description List Of Value Value
Is_RSK_Integrated select Y If you want to integrate the Risk Management application with the Audit Management application. select N If you do not want to integrate the Risk Management application with the Audit Management application. Y
Is_RSM_Integrated select Y If you want to integrate the Resource Management application with the Audit Management application. select N If you do not want to integrate the Resource Management application with the Audit Management application. N
Is_DMS_Integrated select Y If you want to integrate Policy and Document Management System application with the Audit Management application. select N If you do not want to integrate Policy and Document System application with the Audit Management application. N
Is_ISM_Integrated select Y If you want to integrate Issue Management application with the Audit Management application. select N If you do not want to integrate Issue Management application with the Audit Management application. Y
Allow access to documents based on Category-Access Group mapping You can type the following values: Y If you want all the documents to appear in the Select Document window, type this value. N If you want only the production documents to appear in the Select Document window, type this value. Y
Trigger Issues at Monitor Stage If value is Y, the reportable issues that are part of final report will be assigned to Issue owner at Monitor Issue stg and Implement Action form is assigned to Action Owner. If value is N, these issues will be assigned to owner at Manage Issue stg. N

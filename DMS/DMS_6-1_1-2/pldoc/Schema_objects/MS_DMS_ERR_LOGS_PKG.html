INSERT_LOG method is used for inserting messages to the MS_DMS_ERR_LOGS table
Parameters:
PN_ERROR_HANDLE_ID - This will be the numerical value and it will same for one transcation or one single flow
PN_ERROR_SEQUENCE - Numerical value generated by sequence for every log into MS_DMS_ERR_LOGS table
PD_ERROR_TIME - Captures the error time for the debug and error messages
PC_ERROR_TYPE - Captures the string like 'e', 'E', 'DMSLOG'
PC_ERROR_TEXT - Text captured while logging error and debug messages
PN_PID - Process_Instance_id of the submitted form
Throws:
OTHERS - If there is any Error then storing the error details in MS_DMS_ERR_LOGS table.

PROCESS_DOCUMENT_REVIEW method is called for document review flow.
This method will handle the operations from reviewer stage to Production.
This method in turns calls the sub programs for insertion into DMS base tables.
Parameters:
X_PROCESS_INSTANCE_ID - Process_instance_id of the form submitted.
X_CURRENT_LEVEL_NUMBER - Level number or stage id of the form.
X_CURRENT_USER - Current logged in user name or form submitted user.
X_MOVE_ACCROSS_LEVELS - Next stage Level of the form submitted.
X_EMAIL_FLAG - Flag to determine the email checks.
X_RETURN_LEVEL_NUMBER - Stage sequence numbe of the submitted form.
X_USER_ACTION_TYPE - Action type choosen by the submitted user.
X_FINAL_STAGE - Last stage of the form.
X_RET_ARGS1 - Out parameter, Document Action type will be value for this.

This Procedure is used to do the required changes in the policy form after the form is submitted. Create-feedback stage
Parameters:
LT_IN_DATA_ORIG - The Data XML in CLob format.
LT_IN_DATA - The In Data XML in CLob format.
PC_PROCESS_CODE - The workflow code name for the flow.
LN_ORIG_PROCESS_INSTANCE_ID - The process instance identifier for which the form is submitted.
LN_PROCESS_INSTANCE_ID - The process instance identifier for which the form is submitted.
XN_ERROR_HANDLE_ID - The error handle id for which the form is submitted.
XN_ERROR_SEQ - The error sequence for which the form is submitted.
XN_ERROR_CODE - The error code Id for which the form is submitted.
Throws:
OTHERS - If there is any error then storing the error details in MS_DMS_ERR_LOGS table.

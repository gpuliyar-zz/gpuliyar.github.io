GET_INFOCENTER_TAB

public NUMBER GET_INFOCENTER_TAB(I_USER_ID IN VARCHAR2)
Returns the infocenter type for the given user ID I.e. whether the infocenter has to be displayed for Module or for App. This method is called by DMSTabHelper Java code. Return value 0 indicates infocenter should be disaplayed as per the Module need Return value 1 indicates infocenter should be displayed as per the App need

GET_DOCUMENT_ACCESS_FOR_USER

  
GET_DOCUMENT_ACCESS_FOR_USER(I_USER_ID IN VARCHAR2, 
                               I_INFOLET_ID IN NUMBER DEFAULT NULL, 
                               I_PROCESS_INSTANCE_ID IN NUMBER DEFAULT NULL, 
                               I_FORM_FIELD_ID IN NUMBER DEFAULT NULL, 
                               I_DOCUMENT_ID IN VARCHAR2 DEFAULT NULL, 
                               O_RETURN_FLAG OUT NUMBER, 
                               O_RETURN_MSG OUT VARCHAR2, 
                               I_OPERATION_MODE IN VARCHAR2 DEFAULT NULL, 
                               I_OPERATION_PARAM IN VARCHAR2 DEFAULT NULL)
Procedure to check the security for downloading documents.
Parameters:
I_USER_ID - Logged In User Id
I_INFOLET_ID - Form Metric Id
I_PROCESS_INSTANCE_ID - Latest Process Instance Id for the Document
I_FORM_FIELD_ID - column_sequence For the form Field
I_DOCUMENT_ID - Document Object Id for the Document
O_RETURN_FLAG - Return Flag for the access or access denied
O_RETURN_MSG - Return Details
I_OPERATION_MODE - Mode of access whether Download or View Etc
I_OPERATION_PARAM - whether Normal Document or Inline Document
Throws:
NO_DATA_FOUND - o_return_flag := 1 and o_return_msg := 'Access Denied' and Populate error details into MS_DMS_ERR_LOGS table
OTHERS - o_return_flag := 1 and o_return_msg := 'Access Denied' and Populate error details into MS_DMS_ERR_LOGS table

MS_DMS_REMOVE_OOF_USERS

public CLOB MS_DMS_REMOVE_OOF_USERS(USER_ID_STR IN CLOB, 
                                    APPROVAL_TYPE IN VARCHAR2, 
                                    STAGE IN VARCHAR2, 
                                    INITIATOR_NAME IN VARCHAR2)
A subset of the input user ids are returned. Subset is identified based on following conditions:
1. Assigned to users of out of office users are removed from the input
2. If the assigned to user is same for more than one out of office users, then all out off office users except the first are removed from the input
Parameters:
USER_ID_STR - comma separated user id
APPROVAL_TYPE - not used
STAGE - not used
INITIATOR_NAME - not used
Returns:
subset of input user id

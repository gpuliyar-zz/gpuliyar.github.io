#2 : QSM Approver

Description Users who are mapped to the approver role can approve all the survey/scorecard/certifications and request for clarification back to the respondent
Activities
GRC - View All GRC Objects QSM - View All Questionnaire QSM - View Questionnaire

#3 : QSM Control Owner

Description Users who are mapped to control owner role are able to respond to all the survey/scorecard/certifications when a survey is initiated with Applies to as Controls
Activities
GRC - View All GRC Objects GRC - View GRC Object QSM-Control Owner

#5 : QSM Process Owner

Description Users who are mapped to process owner role can respond to a survey/scorecard/certifications when a survey is initiated with Applies To as Processes

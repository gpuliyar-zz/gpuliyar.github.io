#6 : MS_AUD_FIELD_EDITABILITY

Comments
Description This table is to configure the columns that are editable for data correction feature. This can be extended for other features as well where column editability is needed.

# Title
Name		 Type Description
1 Form Name
FORM_NAME 		Varchar2(4000 char) Form Name as available in metric_name column of si_metrics_t table.
2 Column Name
COLUMN_NAME 		Varchar2(4000 char) Column Name as available in result_column_name of si_metric_columns table.
3 Is Editable
IS_EDITABLE 		Varchar2(10 char) This column will have value 'Y' if the field is editable.
4 Feature Type
FEATURE_TYPE 		Varchar2(100 char) This is used to store the feature name where field editability is needed. Example, DC for Data Correction feature.
5 Previous Stage
PREVIOUS_STAGE 		Varchar2(100 char) This is used to store the previous stage of the form.
6 Current Stage
CURRENT_STAGE 		Varchar2(100 char) This is used to store the current stage of the form.
7 Region Name
REGION_NAME 		Varchar2(10 char) Region acronym of the multirow fields as available in multirow_name of si_metric_columns table.
8 ID of Link Report
LINK_REPORT_ID 		Varchar2(100 char) ID of Link Report associated to the LOV/MLOV column

Charts
Name Description
MS_AUD_Chart_Top_5_Audit_Classification This is a stacked bar chart that looks at all audits for the user based on security in the specified time window. We ignore the recurrence frequency in determining if the audit schedule should contribute more than once to the count.

MS_AUD_Chart_Top_5_Checklist This is a bar chart that looks at all audit executions for the user based on security in the specified time window and counts the top 5 most frequently used checklists
MS_AUD_Chart_Top_5_Compliant_Org This is a stacked bar chart that looks at all audits that have published final reports for the user in the specified time window
MS_AUD_Chart_Top_5_NonCompl_Org This is a stacked bar chart that looks at all audits that have published final reports for the user based on security in the specified time window
MS_AUD_Chart_Top_5_NonCompl_Audit_Class This is a stacked bar chart that looks at all audits that have published final reports for the user based on security in the specified time window
MS_AUD_Chart_Top_5_NonComp_Quest This is a stacked bar chart that looks at all audit executions for the user based on security in the specified time window and counts the top 5 most frequently non-compliant questions

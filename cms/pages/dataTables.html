#1 : MS_CMS_VERSION_DTLS

Comments
Description This table is to maintain the version details for each questionnaire

# Title
Name		 Type Description
1 QUE_ID
QUE_ID 		Varchar2(255 char)
2 QUE_NAME
QUE_NAME 		Varchar2(255 char) QUE_NAME
3 VERSION_NUMBER
VERSION_NUMBER 		Varchar2(10 char) VERSION_NUMBER
5 IS_ACTIVE
IS_ACTIVE 		Varchar2(1 char) This column is to capture whether the version is active or not
4 MAPPING_FILE
MAPPING_FILE 		Varchar2(255 char) To store the mapping file and this is based on Questionnaire Id and Version # combination

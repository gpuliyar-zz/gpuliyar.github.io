Introduction

Using the Log Issues form, you can log an issue about the particular alert from the subscribed channel. You can select the issue owner, the selected issue owner gets an assignment to work on and implement the action items for that particular issue.

To log an issue, use the Log Issues form.

F       To access and work on the Log Issues form, the system administrator must assign you to the appropriate organization-role pair combination.

Applicable Roles

·         GRC Feed User

·         GRC Feed Administrator

Prerequisites

·         One or more alerts These are email updates of the latest relevant results (web, news, etc.) based on your choice of query or topic. from channels

Accessing The Log Issue Form

To access the Log Issues form, perform the following steps:

1.       Click the View/Manage Alerts link in the My Forms infoport All related user objects, which are grouped in a single section of the infocenter, that facilitate work of the GRC Alerts infocenter One common and user specific page that appears to users once they login to the application .

The View/Manage Alerts form appears.

2.       Click the Show Report button.

The View Manage Alerts report page appears.

To set the selection of the view manage alert reports refer to Procedure To Mange Alerts section.

3.       Click the enveloped shaped icon  in the Log Issue column.

The Log Issue form appears.

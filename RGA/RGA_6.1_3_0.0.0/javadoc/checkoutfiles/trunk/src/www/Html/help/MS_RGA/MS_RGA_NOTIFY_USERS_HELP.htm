Introduction

Using the Notify Users form, you can notify internal and external users about a particular alert.

To notify a user about an alert, use the Notify Users form.

F       To access and work on the Notify Users form, the system administrator must assign you to the appropriate organization-role pair combination.

Applicable Roles

·         GRC Feed User

·         GRC Feed Administrator

Prerequisite

·         None

Accessing The Notify Users Form

To access the Notify Users form, perform the following steps:

1.       Click the View/Manage Alerts These are email updates of the latest relevant results (web, news, etc.) based on your choice of query or topic. link in the My Forms infoport All related user objects, which are grouped in a single section of the infocenter, that facilitate work of the GRC Alerts infocenter One common and user specific page that appears to users once they login to the application .

The View/Manage Alerts form appears.

2.       Click the Show Report button.

The View Manage Alerts report page appears.

To set the selection of the view manage alert reports, refer to the Procedure To Manage Alerts section.

3.       Click the enveloped shaped icon  in the Notify Users column.

The Notify Users form appears.

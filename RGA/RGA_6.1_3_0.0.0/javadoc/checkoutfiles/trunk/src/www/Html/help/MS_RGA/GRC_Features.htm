GRC Features

The GRC Feeds 5.5 application has the following features:

  • Channels for alerts These are email updates of the latest relevant results (web, news, etc.) based on your choice of query or topic. and structured contents

  • The source data for the created channels is fetched from two sources:

  • Email &endash; The JavaMail Application Programming Interface (API) is used to fetch the data from the specific mail alert inbox

  • Really Simple Indication (RSS Most commonly expanded as Really Simple Syndication) is a family of web feed formats used to publish frequently updated works&emdash;such as blog entries, news headlines, audio, and video&emdash;in a standardized format. An RSS document (which is called a "feed", "web feed”, or "channel") includes full or summarized text, plus metadata such as publishing dates and authorship.) &endash; The RSS API is used to fetch the data from the specific RSS feeds.

  • The user can subscribe to alerts from different channels, mark the relevant alerts to follow-up, and review.

  • The user can notify external and internal users

  • Integration with the Issue Management module. The user can initiate an issue for a particular alert and assign the issue to the issue owner or notify the internal and external users.

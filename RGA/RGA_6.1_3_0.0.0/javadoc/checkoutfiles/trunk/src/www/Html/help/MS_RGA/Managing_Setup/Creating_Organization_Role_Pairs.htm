Creating Organization Role Pairs

Use the Create Organization - Role Pairs page to create an organization role pair.

You need to first select the appropriate organization and role that you want to pair.

To select the appropriate organization, click the Search Organizations link. To select the appropriate role, click the Search Roles link. Click the Go button. You have associated an organization with a role.

Note: You can setup an organization with a role provided the scope (organization hierarchy) of the specific role is the same as that of the specific organization's hierarchy level.

Note: If a red X mark appears in the Delete check box, in line with a specific organization role combination, you CANNOT delete that specific organization role pair. The red X mark in the Delete check box appears when there are one or more users assigned to this organization role combination. Unassign the user/s assigned to this organization role combination and then the red X mark in the check box disappears.

To delete an organization role combination, select the check box in the Delete column, in line with a specific organization role combination, and then click the Update button.

Click the Search Organizations link and select the appropriate organization. Similarly, click the Search Roles link and select the appropriate role. Click the Go button. You have associated an organization with a role.

Note: You can setup an organization with a role provided the scope (organization hierarchy) of the specific role is the same as that of the specific organization's hierarchy level.

Note: If there is a red X mark in the check box, in line with a specific organization role combination, you will not be able to delete it. The red X mark in the check box will be displayed when there are one or more users assigned to this organization role combination. Unassign the user/s assigned to this organization role combination and then the red X mark in the check box will NOT be displayed.

To delete an organization role combination, select the check box in the Delete column, in line with a specific organization role combination, and then click the Update button.

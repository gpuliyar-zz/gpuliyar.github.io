Introduction

Using the My Alert Subscriptions form, you can subscribe to alerts These are email updates of the latest relevant results (web, news, etc.) based on your choice of query or topic. from the available channels by entering key words that act as filters to segregate the content from the subscribed channel. If you opt for email notification from the subscribed channel, you receive an email notification after the new updates are available in the channel.

To subscribe to a channel, use the My Alert Subscriptions form.

F       To access and work on the My Alert Subscriptions form, the system administrator must to assign you to the appropriate organization-role pair combination.

Applicable Roles

·         GRC Feed User

·         GRC Feed Administrator

Prerequisite

·         One or more existing active channels

Accessing The My Alert Subscription Form

To access the My Alert Subscriptions form, click the My Alert Subscriptions link in the My Forms infoport All related user objects, which are grouped in a single section of the infocenter, that facilitate work of the GRC Alerts infocenter One common and user specific page that appears to users once they login to the application.

 

Procedure To Subscribe To Alerts

To subscribe to alerts from the channel, perform the following steps:

Step No.

Field Name

Action

1.           

 

Open the My Alert Subscriptions form.

2.           

 

Channel Name

Select the channel name you want to subscribe to.

F       If you select the same channel for the second time, the system displays an error message.

3.           

Filter Keywords

Enter the key words.

F       The key words entered in this field act as a filter. You can enter key words up to 4000 characters separated by commas.

4.           

Notify me by email

To receive an email notification about the updates on the subscribed channel select Yes or else No.

5.           

Add Row link

For every additional channel that you want to subscribe to, click the Add Row link, and repeat Step 2 through Step 7.

6.           

Delete Last Row link

To delete the last added row, click the Delete Last Row link.

7.           

Delete check box

To delete a specific row, select the Delete check box. The selected row appears in a different color. On form submission, the selected row is deleted.

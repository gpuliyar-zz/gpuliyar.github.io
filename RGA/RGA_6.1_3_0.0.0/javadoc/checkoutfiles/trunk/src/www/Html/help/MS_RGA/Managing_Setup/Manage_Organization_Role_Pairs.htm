Manage Organization Role Pairs

Use the Manage Organization Role Pairs page to assign users to one or more organization-role pairs.

To assign users to one or more organization-role pairs, click the pencil icon in the Assign Users column, in line with a specific organization-role pair.

In the Manage Organization Role Pairs page all the existing organizations and role pairs are listed in a tabular format.

To create an organization role pair, click the Create Organization - Role Pair button.

To access the Manage Roles page, click the Manage Roles button.

PR_POPULATE_SUBSCRPTN_RESP

public  PR_POPULATE_SUBSCRPTN_RESP(O_ERROR_CODE OUT NUMBER, 
                                   O_ERROR_DESC OUT VARCHAR2, 
                                   I_SUPPRESS_NOTIFICATIONS IN CHAR)
Processes the unprocessed channel alerts,sends notification and updates the status of processed alerts.
Parameters:
O_ERROR_CODE - Error code
O_ERROR_DESC - Error description
I_SUPPRESS_NOTIFICATIONS - Email suppress flag (No emails, if its 'Y')
Throws:
NO_DATA_FOUND, - if there are no channel records to be processed.
OTHERS - , when any other exception encountered in processing, updates o_Error_Code with error code and o_Error_Desc with error message.
